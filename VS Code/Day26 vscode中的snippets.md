## Day26

## vscode 中的 snippets

### 插件

- JavaScript(ES6) code snippets
- Vue 3 Snippets
- Vue VSCode Snippets

### 常用快捷键

- `i` + `m` + `p` => `import demo from 'demo'`
- `i` + `m` + `d` => `import { demo } from 'demo'`
- `f` + `n`
- `l` + `o` + `g`
- `v` + `u` + `e`
- `a` + `n` + `f` + `n` 箭头函数
- `i` + `i` + `f` + `e` 立即执行函数
- `r` + `p` 返回 promise

### 技巧

- `tab` 键可以切换位置
- 提示消失时用 `command` + `i` 或 `ctrl` + `space`
