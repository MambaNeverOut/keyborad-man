## vscode 搜索

### 全局搜索

- `command` + `shift` + `f` Mac
- `ctrl` + `shift` + `f` windows

#### 上下移动

- `command` + 方向键

#### 指定文件进行搜索

- `command` + `shift` + `j` Mac
- `ctrl` + `shift` + `j` windows

### 搜索变量

- `command` + `t` Mac
- `ctrl` + `t` windows

### 搜索页面内的变量、函数等等

- `command` + `shift` + `o`
- `command` + `shift` + `o` + `:` 按照类别进行分组

### 搜索 vscode 命令

- `command` + `shift` + `p`

### 搜索文件

* `command` + `p`

### 切换文件

* `ctrl` + `tab`
