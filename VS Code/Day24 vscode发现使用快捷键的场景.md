## Day24

## vscode 发现使用快捷键的场景

### 显示/隐藏资源管理器

- `command` + `b`

### 使用 live-server 开启本地服务

- 安装 live-server 插件
- `command` + `i` `command` + `o`
- `command` + `i` `command` + `c`

### 预览 md

- `command` + `k` + `v`
- `command` + `shift` + `v`

### 打开侧边插件列表

- `command` + `shift` + `x`

### 打开文件所在的目录

* `command` + `k` + `r`
* `shift` + `alt` + `r`

### 复制文件路径

* `command` + `k` + `p`
* `shift` + `alt` + `c`
