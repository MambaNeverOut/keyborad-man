## Day34

## 搜索&查找

## 搜索

### `o`显示多功能搜索框

- 直接输入 link
- 历史记录
- 搜索
- 搜索关键字（可自定义配置）
  - `g` google
  - `g` + `h` github
  - `b` 百度
- `O` 打开新标签

### `b` 搜索收藏夹中的链接

- `B` 打开新标签

### `g` + `e` 显示搜索框/编辑当前地址

- `g` + `E` 打开新标签

### `T` 搜索浏览器中的 tab 页并切换

## 查找

### 页面内查找

- `/` + `要搜索的词`
- `n` 跳转到下一个
- `N` 跳转到上一个

## 在 Vimium C 的帮助页面也想要直接使用搜索和查找

- 在选项中勾选：在 chrome://_/_ 页面上运行
- 打开`chrome://flags/`，搜索`extensions-on-chrome-urls`，选择 enabled
- 可能需要重启才能生效
