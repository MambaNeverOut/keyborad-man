## Day38

## debug

### 打断点

- `command` + `b`
- `command` + `shift` + `b` 会把断点内容存在 Breakpoints 中，方便下次使用

### Step over

- `command` + `'`
- `F10`

### Step into

- `command` + `;`
- `F11`

### 停止当前的断点

- `F5`
- `command` + `\`

### 关闭/开启所有的断点

- `command` + `F8`

### 打印出选中的代码

- `command` + `shift` + `e`

### 跳转到函数

- `command` + `shift` + `o`

### 跳转到指定行

- `ctrl` + `g`
